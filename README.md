# Smart Environmental Monitor

## Project Overview
This project monitors temperature and humidity using an ESP32 and DHT22 sensor in the Wokwi simulator.

## Components Used
- ESP32
- DHT22 Temperature and Humidity Sensor
- Wokwi Simulator

## Features
- Real-time temperature monitoring
- Real-time humidity monitoring
- Serial Monitor output
- IoT-based environmental monitoring

## Circuit Diagram
The DHT22 sensor is connected to the ESP32 and sends temperature and humidity readings.

## Output
The ESP32 continuously reads data from the DHT22 sensor and displays it in the Serial Monitor every 2 seconds.
